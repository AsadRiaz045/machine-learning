# AutoEye Heart AI Diagnostics 🩺❤️

AutoEye Heart AI is a professional Clinical Decision Support System designed for the early detection of heart disease. This project leverages an ensemble of **15 different Machine Learning and Deep Learning models** to analyze patient health parameters and provide real-time diagnostic insights.

## 🚀 Live Demo
Experience the application live here:
https://machine-learning-xf7jxjjvvwrw3ok8jxpz8f.streamlit.app/

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **ML/DL Libraries:** Scikit-Learn, TensorFlow/Keras, Pandas, NumPy
* **Deployment:** Streamlit Cloud

## 🧠 Models Included
The system performs a comparative analysis using 15 distinct algorithms, including:
* **Ensemble Methods:** Random Forest, Gradient Boosting, Bagging Classifier, Extra Trees.
* **Linear & Probabilistic Models:** Logistic Regression, Naive Bayes (Bernoulli), Perceptron, Linear Discriminant Analysis (LDA).
* **Deep Learning:** LSTM (Long Short-Term Memory) network for sequential pattern recognition.
* **Support Vector Machines:** SVC (Linear & RBF Kernels).
* **Tree-based Models:** Decision Trees, K-Nearest Neighbors (KNN).
